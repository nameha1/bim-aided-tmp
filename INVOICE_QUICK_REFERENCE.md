## Common Tasks

### Create New Invoice
1. Click "Create Invoice"
2. Select company profile
3. Enter client details
4. Add items
5. Review total
6. Save & Download

### Edit Existing Invoice
1. Go to Invoices tab
2. Click edit icon (pencil)
3. Make changes
4. Save

### Download Invoice PDF
1. Go to Invoices tab
2. Click download icon
3. PDF downloads automatically

### Search Invoices
- Use search bar
- Search by: invoice #, client name, email, status

### Set Default Company/Bank
1. Go to Company/Bank tab
2. Find the profile
3. Click "Set as Default"
